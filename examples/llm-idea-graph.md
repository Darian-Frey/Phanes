# LLM Idea Graph

Use embeddings to surface "near this" notes that share no tags and link to
nothing, but are clearly about the same thing. A second, opt-in enrichment that
sits beside the extraction model — it proposes adjacency, it never asserts it.

Dormant until the deterministic core and the first enrichment pass are solid;
semantic search is the wrong thing to chase first.

See the [spatial canvas](spatial-canvas.md) for how these proposed neighbours
might actually render.
